# Number Guessing Game (Python + C)

This project contains a **Number Guessing Game** implemented in **two programming languages**:

- **Python**
- **C**

This project is created to demonstrate basic concepts such as random number generation, loops, and conditional logic.

---

## Project Structure

```
Number-Guessing-Game/
│── python_version/
│   └── guess.py
│
│── c_version/
│   ├── guess.c
│   └── guess.exe
│
└── README.md
```

---

## How to Run (Python Version)

1. Navigate to the folder:
   ```
   cd python_version
   ```
2. Run the script:
   ```
   python guess.py
   ```

---

## How to Run (C Version)

1. Navigate to the folder:
   ```
   cd c_version
   ```
2. Compile the program:
   ```
   gcc guess.c -o guess
   ```
3. Run it:
   ```
   ./guess
   ```

---

## Game Rules

- The computer randomly selects a number.
- You must guess the number.
- After each guess, the program tells you:
  - **Too High**
  - **Too Low**
- The game continues until you guess the correct number.

---

## Features

- Random number generation
- Loops and conditional statements
- Beginner-friendly implementation
- Available in both Python and C
- Simple and clear project for students
